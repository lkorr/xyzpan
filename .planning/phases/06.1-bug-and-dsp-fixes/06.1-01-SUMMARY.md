## Decisions Made

- `kDistGainRef=1.0` makes the default source position (Y=1, X=0, Z=0) produce unity gain — physically correct reference
- `kDistGainMax=2.0` (+6dB) caps close-source boost to prevent explosion with floor bounce + reverb accumulation
- Doppler smoother time constant raised from 30ms to 150ms — gives audible, smooth pitch glide rather than zippering
- Per-sample `kDopplerMaxDeltaSamp=2.0` clamp prevents extreme pitch artifacts when jumping large distances in one block
- Dev panel z-order fix: JUCE renders children in add order (last = topmost); glView_ must be added before devPanel_
- LFO strip 2-row layout chosen over widening columns — keeps existing 100px column width, avoids layout overflow
