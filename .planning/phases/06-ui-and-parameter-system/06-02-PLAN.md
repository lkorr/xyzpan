`renderOpenGL()`:
- Clear with background color #1a1108
- Update viewMatrix_ from camera_.getViewMatrix()
- Compute projMatrix_ via glm::perspective(45 degrees, aspect, 0.1, 100.0)
- Read current position from bridge_.read()
- **Read R from processor to scale the room geometry (per user decision: "grid/room wireframe scales with R"):**
  ```cpp
  const float r = proc_.apvts.getRawParameterValue(ParamID::R)->load();
  const glm::mat4 roomModelMatrix = glm::scale(glm::mat4(1.0f), glm::vec3(r));
  ```
  Pass `roomModelMatrix` as the `model` uniform when drawing both the room wireframe and the floor grid.
  The source node model matrix remains identity (source position is already R-scaled in PositionBridge by processBlock).
- Compute source node opacity: 0.1 + 0.9 * (1.0 - distance/sqrt(3)) — full opacity at origin, 10% at max distance
- Compute source node size: perspective-projected apparent size based on camera distance to source
- Draw room wireframe (line shader, model=roomModelMatrix, opacity 0.7, color #8B5E2E)
- Draw floor grid (line shader, model=roomModelMatrix, opacity 0.3, color #3D2A10)
- Draw listener node at origin (sphere shader, model=identity, color #C8A86B, size ~0.06 world units)
- Draw source node at (x, y, z) from bridge snapshot (sphere shader, model=identity, color #E8C46A, opacity computed above)
- Store projected source screen position for hit-testing

`mouseDown()`:
- Recompute projection of source position to screen using current projMatrix_/viewMatrix_
- If within ~12 pixels of projected position: isDraggingSource_ = true, change cursor to drag cursor
- Otherwise: camera orbit drag starts

`mouseDrag()`:
- If isDraggingSource_: compute view-plane delta, un-project to world XYZ, post APVTS write via MessageManager::callAsync
- Else: camera_.applyMouseDrag(dx, dy), repaint

`mouseUp()`: isDraggingSource_ = false, restore cursor

`openGLContextClosing()`: delete VAO/VBO handles, reset shader program pointers

IMPORTANT anti-patterns to avoid (per RESEARCH.md):

- Never write APVTS parameters from GL thread — always use callAsync
- Compile shaders only in newOpenGLContextCreated(), not per-frame
- Create VBOs only in newOpenGLContextCreated(), update uniforms per-frame
