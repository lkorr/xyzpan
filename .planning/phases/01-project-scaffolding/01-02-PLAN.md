    toSpherical(float x, float y, float z):
    1. Hard clamp x, y, z to [-kMaxInputXYZ, kMaxInputXYZ] using std::clamp
    2. Compute distance via computeDistance(x, y, z) (after clamping)
    3. Compute horizontalMag = sqrt(x*x + y*y)
    4. Compute elevation = atan2(z, horizontalMag) -- true spherical elevation per locked decision
    5. Compute azimuth = atan2(x, y) -- Y-forward convention per locked decision. At origin (dist <= kMinDistance), azimuth defaults to 0.0f (front by convention)
    6. Return SphericalCoord{azimuth, elevation, distance}

    computeDistance(float x, float y, float z):
    1. Hard clamp x, y, z to [-kMaxInputXYZ, kMaxInputXYZ]
    2. Compute raw = sqrt(x*x + y*y + z*z)
    3. Return std::max(raw, kMinDistance) -- clamp to minimum distance

    Only use <cmath>, <algorithm> -- no JUCE headers.
