## Decisions Made

- The angular smoother IIR is advanced once per block. The 5ms smoothing constant is far longer than the 1.45ms block duration at 64 samples/44.1kHz, so the difference in convergence is inaudible.
- XZ/YZ orbit LFO cos/sin remain per-sample because `orbitRawXZ`/`orbitRawYZ` change every sample (they are LFO outputs, not parameter knobs).
- `kPI_early` and `kTwoPI_early` are named with the `_early` suffix to avoid redeclaring constexpr `kPI`/`kTwoPI` that are declared later in the same `process()` function scope (in the per-block coefficient block).
- Performance test budgets: 10% CPU for mono path, 20% for stereo path (allowing for dual pipeline cost).
