    **FeedbackCombFilter.h** — Create in `engine/include/xyzpan/dsp/`. Header-only, namespace `xyzpan::dsp`.
    - Power-of-2 ring buffer with bitmask wraparound (same pattern as FractionalDelayLine)
    - `prepare(int capacitySamples)`: allocate next power-of-2 >= capacity + 2, set mask
    - `reset()`: fill buffer with 0.0f, reset writePos
    - `setDelay(int delaySamples)`: clamp to [1, bufferSize - 2]
    - `setFeedback(float g)`: HARD CLAMP to [-0.95f, 0.95f] via std::clamp (DEPTH-04)
    - `process(float x)`: y[n] = x[n] + feedback * buf[readPos]; buf[writePos] = y; return y
    - Integer delay only (no fractional — comb filters don't need it)
    - Reference implementation in 03-RESEARCH.md Pattern 1

    **SVFFilter.h** — Create in `engine/include/xyzpan/dsp/`. Header-only.
    - Parallel class to SVFLowPass (do NOT modify SVFLowPass — Phase 2 uses it)
    - `enum class SVFType { LP, HP, BP, Notch }`
    - `setType(SVFType)`, `setCoefficients(float cutoffHz, float sampleRate, float Q = 0.7071f)`
    - `process(float v0)`: same TPT SVF state update as SVFLowPass, then switch on type for output mix
    - `reset()`: zero ic1eq_, ic2eq_
    - Cutoff clamped to 0.45f * sampleRate before computing g (same stability guard as SVFLowPass)
    - Reference implementation in 03-RESEARCH.md Pattern 2

    **BiquadFilter.h** — Create in `engine/include/xyzpan/dsp/`. Header-only.
    - `enum class BiquadType { PeakingEQ, HighShelf, LowShelf }`
    - `setCoefficients(BiquadType type, float freqHz, float sampleRate, float Q, float gainDb)`: Audio EQ Cookbook formulas (A = 10^(dB/40), alpha = sin(w0)/(2Q))
    - `process(float x)`: Direct Form II (y = b0*x + z1; z1 = b1*x - a1*y + z2; z2 = b2*x - a2*y)
    - `reset()`: zero z1_, z2_
    - CRITICAL: update coefficients per-BLOCK not per-sample (document in header comment)
    - Reference implementation in 03-RESEARCH.md Pattern 3

    **OnePoleLP.h** — Create in `engine/include/xyzpan/dsp/`. Header-only.
    - Distinct from OnePoleSmooth (this uses cutoff Hz, not smoothing time constant)
    - `setCoefficients(float cutoffHz, float sampleRate)`: a = exp(-2*pi*cutoffHz/sampleRate), b = 1-a
    - `process(float x)`: z = b*x + a*z; return z
    - `reset()`: z = 0
    - This is for the ELEV-03 chest bounce 6dB/oct LP at 1kHz
    - Reference implementation in 03-RESEARCH.md "Additional DSP Class Needed: OnePoleLP"
